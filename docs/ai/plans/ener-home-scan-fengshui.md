# Ener Home Scan — ดูฮวงจุ้ยจากรูปถ่าย (แผนเต็ม)

> สถานะ ณ 20 ก.ค. 2026: **สังเคราะห์เสร็จ รอกบเคาะทางเลือก A/B/C ท้ายไฟล์ — ยังไม่ได้สร้าง**
> ที่มา: ไอเดียกบ 19 ก.ค. → ถาม AI 6 ตัว (ไทย 4 + จีน 2 รวมเจาะตลาดจีน) → สังเคราะห์โดย Claude

## 1. คอนเซปต์
ลูกค้าส่งรูปห้องทาง LINE → AI วิเคราะห์ตามหลักฮวงจุ้ยจีน → คะแนนรายห้อง + จุดดี/จุดควรแก้ +
วิธีแก้ที่ทำได้จริง (ย้ายของ เพิ่มม่าน ไม่ต้องรีโนเวท) → รายงานหน้าเว็บสวยแบบเดียวกับรายงานพระ
(ใช้ renderer + ระบบจ่ายเงินในแชทที่มีอยู่แล้ว)

## 2. ขอบเขตที่ AI ทำได้จริง (ฉันทามติ AI ทุกตัว — ข้อสำคัญที่สุด)
- ทำได้แค่ **"ศาสตร์รูปทรง-การจัดวาง" (Form School / 形法)**: ตำแหน่งเตียง/โต๊ะ/เตา
  คานทับ กระจกสะท้อน ความรก แสง ทางเดินถูกขวาง มุมแหลม — จากรูปทำได้แม่น
- **เรื่องทิศ/ดาวเหิน/แปดทิศ (理气) ทำจากรูปไม่ได้เด็ดขาด** (AI ไม่รู้ทิศเหนือจากรูป)
  → ระบบต้องซื่อสัตย์เรื่องขอบเขต ไม่งั้นเสียชื่อ · เฟสหลังค่อยรับทิศจากเข็มทิศมือถือ
- ปีเกิดไม่ต้องถามในเฟสแรก (Form School ไม่ใช้)

## 3. สถาปัตยกรรม 3 ชั้น (โครงเดียวกับระบบจำแนกธูปที่ทำสำเร็จแล้ว)
1. **Vision AI** อ่านข้อเท็จจริงจากรูป (เห็นอะไร อยู่ตรงไหน) — ห้ามตีความเอง
2. **Rule Engine** เทียบ**คลังกฎ structured ~60 ข้อ** ต่อกฎ:
   `เงื่อนไขที่มองเห็นได้ → ผลตามหลักฮวงจุ้ย → วิธีแก้หลายระดับ → ความรุนแรง → rule ID`
   — **ไม่มี open source ในโลก ต้องสร้างเอง = moat ของเรา** · ทุกข้อสรุปในรายงาน trace กลับ rule ID ได้
3. **LLM** เรียบเรียงภาษาจากกฎที่ match เท่านั้น ห้ามมโน

ตัวอย่างกฎ: เตียงตรงแนวประตู (เห็นจากรูป) → ชี่พุ่งรบกวนการนอน (ความรุนแรง: กลาง)
→ แก้: ขยับเตียงเยื้องแนว / ฉากกั้น / ม่าน

## 4. คะแนน
- **5 มิติต่อห้อง**: การไหล-ทางเดิน · ตำแหน่งหลัก+แรงพิง · ความขัดแย้งเฉพาะห้อง (คาน/กระจก/ไฟ-น้ำ)
  · แสง-ความโปร่ง · ระเบียบ-สมดุล
- **แยก "คะแนนความมั่นใจ" ออกจากคะแนนฮวงจุ้ย** — รูปมืด = ลดความมั่นใจ/ขอรูปใหม่
  ไม่ใช่ตัดคะแนนบ้านลูกค้า
- คะแนนรวมบ้านถ่วงน้ำหนัก (ห้องนอนสูงสุด ~30%) + จัดอันดับ "แก้ห้องไหนก่อน"
  ถ่วงด้วยความง่ายในการแก้ (ขยับโต๊ะ 20 ซม. มาก่อนรื้อบิลต์อิน)
- Flow ลูกค้า: ถามแค่ 3 ข้อปุ่มกด (ประเภทสถานที่ → เลือกห้อง → ส่งรูป 2-3 มุม)

## 5. บทเรียนตลาดจีน (ตัวเปลี่ยนเกม)
- **เงินจริงไม่ได้อยู่ที่ค่ารายงาน — อยู่ที่ขายของ**: จีนพิสูจน์แล้ว
  "วิเคราะห์ถูก/ฟรี → ชี้จุด → ขายของแก้ (วัตถุมงคล/หิน/ของแต่งบ้าน)" margin ~80%,
  เคส Douyin 18k followers ขาย 41,000 ชิ้นใน 2 เดือน, GMV วันเดียว 1.2 ล้านหยวน
  → **เราได้เปรียบจีน**: ขายพระในไทยถูกกฎหมาย+เป็นวัฒนธรรม + มีแผน Ener Bridge รออยู่
  → รายงานฮวงจุ้ย = เครื่องชงการขาย ไม่ใช่ตัวสินค้า
- **รายงานบาง = ตาย**: template ซ้ำ ๆ repeat <8% ลูกค้าด่า "จ่าย 99 ได้ 3 ย่อหน้า" ·
  รายงานหนา (หมื่นคำ $39.9) repeat 38.7% — เรามีเครื่องทำรายงานเว็บสวยพิสูจน์แล้ว
- **ยังไม่มีเจ้าตลาด** "ถ่ายรูปห้อง→AI วิเคราะห์" แม้แต่ในจีน (ที่ดังคือสายดวง/โหงวเฮ้ง) — สนามว่าง
- จีนแผ่นดินใหญ่โดนกวาดล้าง 2025 (ปิด 3500 แอป, WeChat แบน) — ไทยไม่มีปัญหานี้
- ภาษา: ไม่ขู่ ไม่ฟันธงชะตา — "ตามหลักความเชื่อ...อาจทำให้...แนะนำปรับ" (กรอบเดิมของเราใช้ได้เลย)
- เครื่องมือยืมได้: floor plan recognition (Structify-AI open source, 华为云 API) —
  แต่เฟสแรกใช้ multimodal LLM อ่านรูปตรง ๆ พอ · ไม่ต้องใช้เข็มทิศเพราะทำ Form School

## 6. เศรษฐศาสตร์ (คำถามกบ: ลูกค้าคนนึงใช้กี่ครั้ง คุ้มไหม)
- ขายรายงานอย่างเดียว: LTV ~300-450 บาท/คน/ปี แล้วแผ่ว (บ้านมีหลังเดียว จ่ายซ้ำต่ำโดยธรรมชาติ)
  ต้นทุน AI ~2-5 บาท/บ้าน (margin 95%+) แต่เป็น one-shot
- **ความคุ้มจริงอยู่ 3 ทาง**:
  1. เปิดตลาดใหม่กว้างกว่าคนเล่นพระ (คนมีบ้าน/คอนโด/ร้าน) — ยิงแอดดึงเข้า OA แล้วขายสแกนพระต่อ
  2. จ่ายซ้ำแท้จริง: **ร้านค้า/ออฟฟิศ** (จัดบ่อย จ่ายแพง — ซินแสจริงเริ่ม 5,000) +
     **"ได้พระใหม่มา วางมุมไหนดี"** (ผูกวงจรสแกนพระ = repeat จริง)
  3. ประตูป้อนการขายวัตถุมงคล (บทเรียนจีนข้อใหญ่)
- repeat สร้างจาก "ฉากใหม่": ย้ายบ้าน/เปิดร้าน/ตรวจให้พ่อแม่-เพื่อน/ปีใหม่-ตรุษจีน
- **ไม่ทำรายเดือน** (AI 2 ตัวเบรกตรงกัน: คนไม่จัดบ้านใหม่ทุกเดือน churn แน่) →
  ใช้ "Recheck Pass 90 วัน" แทน (แก้แล้วส่งรูปมาเช็คซ้ำ — ตัว retention ที่คู่แข่งไม่มี)

## 8. เรื่องต้องทำจริงจังตั้งแต่วันแรก
- **Privacy**: รูปในบ้าน sensitive กว่ารูปพระมาก — เตือนเก็บเอกสาร/บัตรก่อนถ่าย +
  นโยบายลบรูป + ไม่เอารูปไปเทรน
- เกณฑ์ปฏิเสธ: รูปมืด/แคบ/ไม่เห็นประตู → ขอรูปเพิ่มแบบสุภาพสั้น ๆ · พลาด 3 รอบ →
  ปุ่ม [ให้อาจารย์ดูเอง] (เปลี่ยนวิกฤตเป็นขาย premium)
- ชื่อบริการ: **Ener Home Scan** — ขายว่า "ตรวจจุดจัดวางจากภาพ พร้อมวิธีแก้ที่ทำได้จริง"
  ไม่ขายว่า "AI แม่นกว่าซินแส"
