# Ener on WhatsApp — แผนตลาดต่างชาติ (สิงคโปร์เป้าแรก)

> สถานะ ณ 21 ก.ค. 2026: แผนเคาะร่วมกับกบแล้ว · กบกำลังไปจดทะเบียนพาณิชย์บุคคลธรรมดา
> ที่มา: เคสลูกค้า James Ho (สิงคโปร์ ทักมา organic) + วิเคราะห์ตลาด + คำตอบ AI เรื่องช่องทาง

## ทำไมตลาดนี้
- ตลาดพระเครื่องไทยรวม 40,000+ ล้านบาท/ปี (กสิกร 2019: 17-23k ล้านเฉพาะคนไทย ไม่รวมต่างชาติ)
- ตลาดต่างชาติถูกครองโดย **สิงคโปร์+มาเลเซีย** มาแต่แรก มีร้านของคนสองชาตินี้หลายร้อยร้านในตลาดพระไทย
- ราคาพระในสิงคโปร์เริ่ม ~139 SGD ถึงหลายพัน SGD/องค์ → ค่าอ่าน $3-5 = เศษเงิน
- Pain จริง: ซื้อของแพงแต่อ่านไทยไม่ออก ไม่รู้พุทธคุณ ไม่รู้เข้ากับตัวเอง
- ยังไม่มีใครทำ "AI อ่านพลังพระไทยภาษาอังกฤษ/จีน" จริงจัง — สนามว่าง
- หลักฐานสด: James Ho หาเรามาเจอเองโดยไม่มีแอด

## หลักการช่องทาง: "ไปอยู่ในแอปที่ลูกค้าเปิดทุกวัน"
| ตลาด | บ้านของลูกค้า | สถานะ |
|---|---|---|
| ไทย | LINE OA | ✅ มีแล้ว |
| ไต้หวัน/ญี่ปุ่น | LINE | ✅ ใช้ OA เดิม ยิงแอดได้เลย (รอ funnel EN/จีน) |
| **สิงคโปร์/มาเลย์/ฮ่องกง/ตะวันตก** | **WhatsApp** | 🔨 แผนนี้ |
| จีน | WeChat | เฟสท้าย (กำแพงสูง) |

Web = ประตูรับแขก (กดจากแอด/Google เข้าได้ทันที ไม่ต้องโหลดแอป) — **ไม่ใช่ที่เก็บลูกค้า**
(จุดอ่อน web ที่กบชี้: คนลืม ไม่เหมือน LINE) → สมอ 2 ตัว: **email** (รายงาน+weekly pick)
+ **ปุ่ม "Chat with Ajarn on WhatsApp"** ท้ายรายงาน (wa.me — เปิดแอปที่มีอยู่แล้ว)

## Funnel
FB Ads + Google → Web Scan (อัปโหลดรูป+วันเกิด+email, ฟรีครั้งแรก) → รายงาน EN (มีแล้ว)
→ จ่ายบัตรครั้งต่อไป ($2.99/4.99/19.99 — แพงกว่าราคาไทย 3-5 เท่า สมเหตุผลกับตลาด)
→ email + WhatsApp เป็น retention

## งานสร้าง (เรียงลำดับ)
0. **กบ**: จดทะเบียนพาณิชย์บุคคลธรรมดา (50฿ ที่เขต — ระบุประเภท "พาณิชย์อิเล็กทรอนิกส์/
   บริการข้อมูลและวิเคราะห์ภาพผ่านอินเทอร์เน็ต" เลี่ยงคำว่าดูดวงเพียว ๆ กันติดหมวด high-risk)
   + ซื้อซิมใหม่ 1 เบอร์ (ไม่เคยผูก WhatsApp)
1. **Web Scan MVP** (~2-3 วัน): หน้าอัปโหลด+วันเกิด+email → enqueue เข้า pipeline สแกนเดิม →
   รายงาน EN เดิม · identity ใหม่นอก LINE (web user + email) · ฟรีครั้งแรก
2. **รับบัตร** (~1 วัน): **GB Prime Pay** เป็นหลัก (รับบุคคลธรรมดา+ทะเบียนพาณิชย์ ค่อนข้างชัวร์,
   บัตรต่างชาติ+PromptPay, ~2.9-3.65%) · Omise = สำรอง (ช่วงหลังเน้นนิติบุคคล ต้องเช็คตอนสมัคร) ·
   Lemon Squeezy = แผน C (MoR ไม่ต้องมีทะเบียน เงินเข้า Payoneer ที่กบมี, ~5%) ·
   Stripe = ไว้เมื่อมีนิติบุคคล ตปท. · webhook จ่ายสำเร็จ → markPaymentApprovedAndUnlock เดิม (อัตโนมัติ ไม่ต้องตรวจสลิป)
3. **English funnel pack บน LINE เดิม** (~1 วัน): greeting/ขอวันเกิด/รับรูป/paywall สองภาษา
   ตาม detectReplyLanguage (ทำแล้วฝั่ง consult) — รับ James-type + เปิดทางยิงไต้หวัน
4. **WhatsApp Cloud API** (~2-3 วัน): อะแดปเตอร์ webhook รับ-ส่ง (สมองแชท/สแกน/ตัวตรวจ ใช้ตัวเดิม
   ทั้งหมด) · identity = เบอร์โทร · การ์ด Flex → render PNG (resvg มีแล้ว) + ปุ่ม ≤3 ·
   **เริ่มแบบ unverified ได้เลย** (user-initiated ไม่จำกัด, business-initiated ~250/วัน) →
   verify เต็มด้วยทะเบียนพาณิชย์ · **AI ตอบแชทถูกนโยบาย Meta** (chatbot เป็น use case หลักของแพลตฟอร์ม
   ธุรกิจสายความเชื่อไม่อยู่ในลิสต์ต้องห้าม) · ข้อจำกัด: push เกิน 24 ชม ต้อง template อนุมัติ+จ่าย/ข้อความ
   (~1.5-3฿ ในสิงคโปร์) → push หลักใช้ email (ฟรี), WhatsApp template เฉพาะที่คุ้ม
5. **แอดทดสอบสิงคโปร์**: งบ ~2,000฿/4-5 วัน รูปแอด EN ที่ทำแล้ว (scratchpad/ads) ·
   targeting: สิงคโปร์ตรง + expat ในไทย (ภาษาอุปกรณ์ EN/ZH + interest Buddhism/Amulet) ·
   วัด: คลิก → เริ่มสแกน → จ่าย → ตัดสิน scale
